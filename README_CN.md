# 求职辅助智能体 (Job Hunt Agent)

**求职辅助智能体 (Job Hunt Agent)** 是一款基于先进 AI 技术的求职辅助与决策支持“神器” 🪄。它旨在帮您彻底告别繁琐的求职内耗，一站式管理申请、追踪经验、透视职位，并为您 **量身定做🔥** 最吸睛的简历与求职信！

*   😵 **阅读招聘信息，又长又累？字多到眼花？**

    👉 **企业底裤级背调！** 15秒为您清晰扒出行业背景、潜在风险，防火防盗防忽悠！🛡️

    👉 **雷达级岗位匹配度分析！** 多维信息直观展示您与岗位的契合度，弹无虚发，有的放矢！🏹
*   🤯 **每次投递都要针对性改简历，折腾到头秃？**

## 🌟 核心闪光点 (Core Features)

- 🧠 **硬核职位解析**：像顶级猎头一样拆解职位描述，完美对齐您的过往履历。
- 📦 **个人经验智库**：妥善存储并升华您的工作数据，再也不怕丢三落四，让经验活起来。

## 环境要求

- **Python 3.10+**
- **Node.js 18+** 及 npm

## 安装与配置

### 自动安装 (Windows)

项目根目录提供了一个自动化脚本，帮助您快速配置 Python 与 Node 环境。

1. 在项目根目录打开 PowerShell。
2. 运行脚本：
   ```powershell
   .\run_locally.ps1
   ```
3. 打开生成的 `backend/.env` 文件并填入您的 API Key（如 `PHARSE_LLM_API_KEY`, `EMBEDDING_API_KEY` 等）。

### 手动安装

#### 1. 后端设置

```bash
cd backend
python -m venv venv
# 激活虚拟环境
# Windows: .\venv\Scripts\activate
# Mac/Linux: source venv/bin/activate
pip install -r requirements.txt
```

在 `backend` 目录下创建 `.env` 文件（添加必需的环境变量，例如 `DATABASE_URL`）。

#### 2. 前端设置

```bash
cd frontend
npm install
```

## 如何使用

### 搭配浏览器插件使用

为了体验完整的求职辅助功能，您需要配合项目提供的浏览器插件（位于 `/jobs` 目录）来摄入并在页面上获取职位及简历信息：

1.  在您的浏览器中打开扩展管理页面（例如 Chrome/Edge 浏览器中的 `chrome://extensions/`）。
2.  开启**开发者模式**。
3.  点击**加载已解压的扩展程序**，并选择本项目根目录下的 `jobs` 文件夹。
4.  安装完成后，您可以在支持的招聘网站上使用该插件快速抓取职位信息，并发送至求职智能体中。

### 启动应用程序

您需要在不同的终端分别启动后端和前端服务。

**终端 1 (后端):**
```bash
cd backend
# 如果还未激活，请激活虚拟环境
uvicorn app.main:app --reload --port 8004
```

**终端 2 (前端):**
```bash
cd frontend
npm run dev
```

最后，在浏览器中打开 `http://localhost:3004` (或由 Vite 指定的运行端口)。
